# Airbnb Market Analysis Dashboard

## Overview

This Tableau dashboard analyzes Seattle Airbnb listings to identify pricing trends, revenue patterns, and location-based market opportunities. The project demonstrates how property size and geographic location influence Airbnb pricing and performance.

## Tools Used

* Tableau Public
* Microsoft Excel

## Dashboard Highlights

* Average Price by Bedroom
* Revenue Trend Analysis
* Price Analysis by Zip Code
* Interactive Geographic Map
* Bedroom Listing Distribution
* Dynamic Filters and Business Insights

## Key Insights

* 6-bedroom listings achieve the highest average prices.
* Most Airbnb inventory consists of 1-bedroom properties.
* Premium zip codes command significantly higher nightly rates.
* Revenue shows a positive trend throughout the year.
* Property size and location are key drivers of pricing.
